# Number Guessing Game

A simple Java console-based game where the user tries to guess a randomly generated number within a given range.

## 📌 About
This project was built during my internship at decodeLabs as a practice project to strengthen my understanding of Java fundamentals like classes, objects, loops, and conditionals.

## 🎮 How it Works
- The game picks a random number between a min and max value (e.g. 1–100)
- The user keeps guessing until they get it right
- The game tracks the number of attempts and gives feedback (too high / too low)
- After each round, the user can choose to play again

## 🛠️ Built With
- Java
- IntelliJ IDEA
